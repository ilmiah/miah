About me

Setelah beberapa tahun di malaysia saya di perkenalkan EUB oleh seorang teman lama, kenapa saya sebut teman lama? karena dialah teman saya satu-satunya di sini yang dari indonesia, apa itu EUB? EUB ( edukasi untuk bangsa ) adalah sarana belajar yang di sediakan oleh pemerintah secara gratis untuk para TKI di sini supaya selain bekerja menjadi buruh migran, TKI juga dapat menimba ilmu yang dapat di manfaatkan ketika kembali ke kampung halaman masing-masing nantinya, sarana belajar yang di sediakan oleh EUB yaitu: computer, bahasa inggris, coding, menjahit, kraff & wirausaha.

Semenjak mengenal EUB hujung minggu saya habiskan waktu di EUB, semua kelas saya ikuti secara bergantian dari batch 8-12 saat ini saya sedang belajar coding yang baru di adakan oleh EUB 2 batch terakhir, walaupun saya tidak tahu apa itu coding namun saya ingin mencoba & belajar karena di dorong rasa penasaran & rasa ingin tahu yang tinggi hihi... sebelum kemudian saya mengenal MS (Mandiri Sahabatku), MS yaitu seminar atau pelatihan intepreneur utuk para TKI di beberapa negara termasuk malaysia, dari MS saya dapat mengenal codingmum, codingmum yaitu kelas coding khusus wanita yang di adakan oleh BEKRAFF ( badan ekonomi kreatif ) untuk melatih para TKW supaya dapat membuat website sendiri, para profesional dari indonesai di kuala lumpur sanggup menjadi sukarelawan untuk berbagi ilmu secara gratis & dengan kesabaran yang tinggi tentunya menghadapi ibu-ibu student di EUB maupun di codingmum ini.
